  <h1>StandAIO</h1>
  StandAIO是GTAV Stand Mod Menu 的All In One多位一体合集  旨在建立并完善一个更好的Modder社区<br><br>
  本项目前身为2021/10由MrLGXC创建的Stand配置，后与MrLGXC在11月创建的lance/wiri等脚本汉化合并。<br><br>
  于2021/12正式开始StandAIO项目<br>
