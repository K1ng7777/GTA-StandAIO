## 注意事项
> **由于版本更迭过多，若遇到代码冲突脚本错误的情况请酌情备份并删除旧文件**<br>
> **优先使用Constructor/LanceSpooner加载自定义载具和地图**<br>
> **在线模式中最多同步80个实体，超出后仅本地显示。若别人无法看到请先清除旧的实体**<br>
> **若使用Menyoo生成载具后异常，请并重载ASI**<br>
> **若使用Menyoo生成地图后黑屏无限加载，请切换战局，并重载ASI**<br>
